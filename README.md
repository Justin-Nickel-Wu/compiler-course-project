#### 项目说明

本仓库用于杭州电子科技大学2025秋季学期编译原理课程实践项目。由于涉及个人信息暂不包含设计文档。

---
#### 功能说明
##### 正则表达式住转换

支持正则表达式至NFA、DFA、最小化DFA的转换与示意图输出。
支持的字符集: a..z
支持的操作符: ( )、*、?、+、|

##### 文法的左递归消除

支持Unicode编码输入与输出。
以Unicode字符作为符号单位。
支持 -> 与 → 两种箭头的识别。

---
#### 使用指南

命令行输入:
- make run DFA: 编译并执行正则表达式至最小化DFA的转换。根据命令行提示完成输入后，示意图将会输出至output/。
- make run RLR: 编译并执行文法的左递归消除。输入文法为./grammar.in
- make: 仅编译。
- make clean: 清除编译结果。
